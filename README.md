# State Machine Visualization WebApp

## Overview

lets you edit state machines programmatically easily hehehehehheee

## Technologies

### Frontend

- React
- Next.js
- Canvas Node Editor

### Backend

- Python
- Python AST Module
- Flask

## Installation & Setup

### Prerequisites

- Node.js (for frontend development)
- Python & Flask (for backend development)
- Firebase account (for authentication and database)

### Steps

1. **Clone the repository:**
   ```sh
   git clone https://github.com/Peter-Dated-Projects/02-28-2025_statemachine-editor
   cd 02-28-2025_statemachine-editor
   ```
2. **Install frontend dependencies:**
   ```sh
   cd frontend
   npm install
   ```
3. **Install backend dependencies:**
   ```sh
   cd ../backend
   pip install -r requirements.txt
   ```
4. **Run the frontend:**
   ```sh
   cd ../frontend
   npm run dev
   ```
5. **Run the backend:**
   ```sh
   cd ../backend
   python app.py
   ```
